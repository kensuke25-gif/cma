【Point Check 1-1】 景気動向指数 ― 先行・遅行系列の識別 テキスト p.4〜9
日本の景気を判断する1つの方法として、内閣府が公表している景気動向指数（先行・一致・遅行系列）の分析がある。景気動向指数の先行系列採用指標と遅行系列採用指標を各々2つずつ挙げなさい。
解答・解説
📌 解くためのポイント
  • 景気動向指数は「先行（Leading）・一致（Coincident）・遅行（Lagging）」の3系列に分類される
  • 先行系列：景気の山・谷に先行して変動する指標。新規求人数・株価など「未来」を映す
  • 遅行系列：景気の山・谷に遅れて変動する指標。常用雇用・法人税収など「結果」を映す
  • DI（景気動向指数）＝改善系列数÷全系列数×100。50%超で景気拡張を示す
解答
先行系列（例）：新規求人数（除学卒）、東証株価指数（TOPIX）、長短金利差、新設住宅着工床面積 など
遅行系列（例）：常用雇用指数（製造業）、完全失業率（逆サイクル）、消費者物価指数変化率、法人税収入 など
分類の考え方

先行系列は「経営者・投資家の期待・計画」を反映する指標が多い（新規採用計画、株価、金利差など）。遅行系列は「実際に雇用・消費として確定した数字」が多い（雇用者数、物価など）。試験では具体的な指標名を複数記憶しておくこと。

【Point Check 1-2】 GDPデフレーター ― 原油輸入価格上昇の影響 テキスト p.11〜14 ｜ 2019年午後 第2問
ある年度、ある国において、原油輸入価格が大幅に上昇したが、国内物価（国内需要デフレーター）や輸出物価は上昇しなかったとしよう。このとき、この国のGDPデフレーター対前年比上昇率はどのような値になるか、GDPデフレーターの算出を踏まえて説明しなさい。
⚠ ひっかけポイント
「輸入物価が上がったのだから物価指数は上がるはず」という直感は誤り。GDPデフレーターの算出式を正確に理解していないと間違える。
解答・解説
📌 解くためのポイント
  • GDPデフレーター＝名目GDP ÷ 実質GDP × 100
  • 名目GDP（支出面）＝ C+I+G+（輸出−輸入）。輸入は控除項目
  • 輸入価格上昇 → 名目輸入額↑ → 名目GDPから控除される額↑ → 名目GDP↓
  • 実質GDPは基準年価格で評価するため、輸入価格変動の影響を受けない（不変）
  • 結果：名目GDP↓ ÷ 実質GDP（不変）＝ GDPデフレーター低下 → 上昇率はマイナス
解答
GDPデフレーターの対前年比上昇率はマイナス（低下）となる。
論理の流れ
原油輸入価格 大幅上昇
名目輸入額 ↑（控除項目が増加）
名目GDP ↓（国内物価・輸出物価は不変）
実質GDP：不変（基準年価格で評価するため）
GDPデフレーター＝名目GDP↓÷実質GDP → 低下（上昇率マイナス）

この現象は「輸入デフレ」とも呼ばれ、実際に2021年以降の日本で観察された。輸入物価が上昇してもGDPデフレーターが低下するという逆説的な動きを理解しておくこと。

【Point Check 1-4】 45度線分析 ― 限界消費性向と財政政策の効果比較 テキスト p.22〜27 ｜ 2021年午後 第2問
【前提】W国・X国の消費関数：C = C₀ + c(Y−T)。X国の限界消費性向 cX ＞ W国の cW（ともに 0＜c＜1）。Z国は限界消費性向の高い・低い家計が各50%ずつ存在。
問1．W国とX国において、定額税が同額減税された場合、それぞれの財政政策の効果の相違を説明しなさい。
問2．Z国において、総額が等しい2つの定額給付方法（α：全家計に一律S円、β：限界消費性向が高い家計のみに2S円）のどちらがGDP増加効果が大きいか答えなさい。
解答・解説
📌 解くためのポイント
  • 減税乗数 ＝ c ÷ (1－c)。c が大きいほど乗数が大きくGDP増加効果が高まる
  • 定額給付 ＝ 定額減税と同等の乗数効果（可処分所得増加の経路が同じ）
  • 「誰に給付するか」が鍵：限界消費性向の高いグループへ集中給付 → 乗数大 → GDP効果大
問1 解答
定額税減税の均衡GDPへの効果は減税乗数 c÷(1−c) に比例する。cX＞cW であるためX国の減税乗数がW国より大きく、同額の減税でもX国のGDP増加はW国を上回る。
ΔY = [c ÷ (1−c)] × 減税額　→　c が大きいほどΔYは大
問2 解答
β（限界消費性向が高いグループのみに2S円給付）の方がGDP増加効果が大きい。
理由：定額給付は定額減税と同等の乗数が働く。同額の総給付でも、乗数の大きい（c が高い）グループへ集中するβの方がΔYが大きくなる。

【Point Check 1-5】 IS-LM分析 ― 景気過熱時の中央銀行による金融引き締め テキスト p.28〜34 ｜ 2019年午後 第1問・2023年午前 第9問
IS-LM分析において、当初の均衡点がE₀であった。その後、家計の消費関数における基礎消費が増加したためIS曲線がIS₀からIS₁へとシフトし、均衡点がE₁に移動した。このような動きは景気過熱による物価上昇を引き起こす可能性がある。
当該国の中央銀行は、GDPをもとのY₀に戻すためにどのような政策を行う必要があるか、IS-LM分析の図を用いて説明しなさい。金利とGDPの動きについても説明すること。
解答・解説
📌 解くためのポイント
  • IS曲線が右シフト → GDP↑・金利↑。これをもとに戻すには LM を左シフトさせる
  • LM曲線左シフトの手段：マネーストック減少（金融引き締め）＝ 売りオペレーション
  • LM左シフトにより金利はさらに上昇するが、GDPはY₀へ戻る
  • 「GDPをY₀に戻す」ことが目標なので、金利変化は副作用として述べること
解答
中央銀行は金融引き締めとしてマネーストックを減少させる（売りオペレーションなど）。これによりLM曲線をLM₀から左方向にシフトさせ、IS₁との交点をE₀の水準のGDP（Y₀）へ戻すことができる。このとき、GDPはY₀に戻るが、金利は当初のr₀よりも上昇（r₂）する。
IS-LMのシフト論理

IS右シフト（E₀→E₁）：基礎消費↑ → 需要↑ → IS右シフト → GDP↑・金利↑

LM左シフト（E₁→E₀'）：マネーストック↓ → 貨幣供給↓ → LM左シフト → GDP↓・金利↑

目標はGDP=Y₀に戻すこと。金利は基礎消費増加の分だけ恒久的に高い水準（r₂）に留まる。

【Point Check 1-6】 IS-LM分析 ― 財政拡張＋金融緩和の同時実施効果 テキスト p.30〜34 ｜ 2013年午後・2018年午前・2020年午前
景気回復のため、政府が国債発行を財源として政府支出を増加させる一方、日本銀行は発行された国債を市中で買い取るなどして市中金利を上昇させないように政策運営を行うとする。このような政策が行われると、当面の経済にはどのような効果がみられると考えられるか、IS-LM分析の考え方を用いて、グラフと文章で説明しなさい。
解答・解説
📌 解くためのポイント
  • 財政拡張だけなら：IS右シフト → GDP↑・金利↑ → クラウディング・アウト発生
  • 日銀が国債買取（買いオペ）：マネーストック↑ → LM右シフト → 金利上昇を相殺
  • 両者同時：IS右シフト＋LM右シフト → GDP大幅増、金利の上昇は抑制
  • クラウディング・アウト効果がゼロになりGDP増加効果が最大化される
解答
政府支出増加 → IS曲線が右シフト（IS→IS'）。中央銀行が国債買取（買いオペ）でマネーサプライを増加 → LM曲線が右下方シフト（LM→LM'）。2つのシフトが重なり、GDPはY₀からY₁へ大幅に増加する。財政政策のみの場合と異なり、金融緩和がクラウディング・アウト効果を相殺するため、GDPの増加幅が大きくなる。
クラウディング・アウト効果の意味

財政拡張のみでは利子率が上昇し民間投資が抑制される（クラウディング・アウト）。金融緩和を組み合わせることで利子率の上昇を防ぎ、民間投資の抑制を打ち消す。この「財政＋金融の協調」はいわゆる「アベノミクス」の基本構造でもある。

【Point Check 1-8】 IS-LM分析 ― 名目利子率のゼロ下限と流動性の罠 テキスト p.35〜38 ｜ 2017年午前 第9問 問4
IS-LM曲線が描かれた図において、LM曲線は名目利子率に「ゼロ下限」がないものとして描かれている。しかし名目利子率に非負制約がある場合、「ゼロ下限」を考慮する必要がある。
(1) 名目利子率の「ゼロ下限」を考慮した場合、経済の均衡はどのようになるか説明しなさい。
(2) (1)で考察した経済状況において、中央銀行が貨幣供給量を増加させるとき、経済の均衡にどのような影響があるか説明しなさい。
⚠ ポイント
「ゼロ下限」＝流動性の罠の典型問題。金融政策が無効になる状況を正確に説明できるようにすること。
解答・解説
📌 解くためのポイント
  • 名目利子率のゼロ下限：利子率は0%以下にはなれない（非負制約）
  • IS曲線がLMの水平部分に交差する → 均衡利子率0%、LM左端が水平になる（流動性の罠）
  • 流動性の罠では：貨幣供給↑ → LM右シフトするが、均衡点は変化しない（金融政策無効）
  • なぜなら利子率は0%のまま変わらず、投資も変わらず、GDPも変化しない
(1) 解答
ゼロ下限を考慮すると、LM曲線は名目利子率0%の水準で水平となる（横軸と重なる）。IS曲線との交点（均衡点E₀）は、均衡利子率0%、均衡GDP=Y₀となる。
(2) 解答
中央銀行が貨幣供給を増加させるとLM曲線はLM₀からLM₁へ右シフトするが、IS曲線との交点（均衡点）は変化しない。名目利子率は0%のまま、均衡GDPもY₀で不変であり、金融政策は無効となる。
流動性の罠のメカニズム

利子率が十分低い（ゼロ近傍）と、人々は将来の利子率上昇（債券価格下落）を予想して、貨幣を無制限に保有しようとする。追加的な貨幣供給はすべて貨幣需要に吸収され、利子率はそれ以上下がらない。利子率が変化しないため投資も変化せず、GDPは不変となる。日本は1990年代後半〜2000年代にこの状態に陥った。

【Point Check 1-15】 成長会計 ― コブ＝ダグラス型生産関数による要因分解 テキスト p.56〜61 ｜ 2011年・2018年午前・2022年午後
1960〜2000年の日本経済についてY = AKαL1-α（コブ＝ダグラス型）の生産関数を推計したところ、下表のような結果となった。この期間の平均実質経済成長率は7.0%であった。
実質経済成長率（ΔY/Y）：7.0%（年率平均）
労働投入増加率（ΔL/L）：1.0%
資本増加率（ΔK/K）　　：10.0%
労働分配率　　　　　　：0.6（∴資本分配率α = 0.4）
① 労働の寄与度、② 資本の寄与度、③ 技術進歩の寄与度を求めなさい。
解答・解説
📌 解くためのポイント
  • 成長会計の基本式：ΔY/Y ＝ ΔA/A ＋ α(ΔK/K) ＋ (1−α)(ΔL/L)
  • α＝資本分配率（＝1−労働分配率）、(1−α)＝労働分配率
  • 技術進歩（TFP）の寄与度＝成長率全体から労働・資本の寄与を引いた残差
  • 労働分配率＝0.6 → 資本分配率α＝1−0.6＝0.4
解答
ΔY/Y ＝ ΔA/A ＋ α(ΔK/K) ＋ (1−α)(ΔL/L)
① 労働の寄与度 ＝ (1−α)×(ΔL/L) ＝ 0.6 × 1.0% ＝ 0.6%
② 資本の寄与度 ＝ α×(ΔK/K)　 ＝ 0.4 × 10.0% ＝ 4.0%
③ 技術進歩の寄与度 ＝ 7.0% − 0.6% − 4.0% ＝ 2.4%
成長会計の意味

この期間の日本の高度成長は、主に資本投入増（4.0%）によるものであった。技術進歩（TFP）の寄与は2.4%と相対的に小さい。これに対し1990年代以降はTFP成長率が低下し、潜在成長率が落ち込んだとされる。

【Point Check 1-16】 新古典派経済成長モデル ― ソロー＝スワンモデルと貯蓄率上昇の効果 テキスト p.62〜67
1人当たり産出量 y＝f(k)（kは資本装備率）、貯蓄率 s、人口成長率 n とする。
(1) このモデルにおける資本装備率の動的変化 Δk を示す基本方程式を、f(k)・s・k・n を用いて示しなさい。
(2) 長期均衡の資本装備率がk₁*である。貯蓄率がs₁からs₂に上昇した場合に生じる変化を説明しなさい（1人当たりGDPへの影響も含む）。
解答・解説
📌 解くためのポイント
  • 基本方程式：Δk ＝ s·f(k) − nk　（投資 − 人口増加による資本装備率の希薄化）
  • 長期均衡（定常状態）：Δk＝0 → s·f(k*)＝nk*
  • 貯蓄率↑ → s·f(k)が上方シフト → 定常状態の資本装備率 k* が増加 → 1人当たりGDP↑
  • ただし技術進歩がなければ、長期的な1人当たりGDP成長率は変化しない（水準が上がるだけ）
(1) 解答
Δk ＝ s·f(k) − nk
左辺：資本装備率の変化。第1項（s·f(k)）：1人当たり貯蓄（＝投資）。第2項（nk）：人口増加による資本装備率の希薄化。
(2) 解答
貯蓄率がs₁からs₂に上昇すると、s·f(k)が上方シフトし、Δk＝0となる定常状態の資本装備率がk₁*からk₂*へと増加する。これに伴い、1人当たりGDP（＝f(k*)）も増加する。

【章末問題 1.1】 45度線分析 ― 財政政策の効果・限界消費性向・定額給付 テキスト p.22〜27 ｜ 2021年午後 第2問
【前提】W国・X国の消費関数：C = C₀ + c(Y−T)。cX ＞ cW（限界消費性向）。Z国は限界消費性向の高低両グループが各50%。総額均等の給付方法：α（全家計にS円）/ β（高グループのみ2S円）
問1．W国とX国において定額税が同額減税された場合、それぞれ財政政策の効果の相違について説明しなさい。
問2．αとβのどちらを採用する方がGDPを増加させる効果が大きいか答えなさい。また理由も説明すること。
⚠ 注意
問2：「定額給付＝定額減税と同等の効果」を前提に論じること。総給付額が等しいときの乗数の違いに着目する。
解答・解説
📌 解くためのポイント
  • 減税乗数 ＝ c ÷ (1−c)　→　c が大きいほど乗数が大きくGDP増加効果が高い
  • 定額給付は可処分所得増加を通じて定額減税と同等の乗数で作用する
  • 問2：同額総給付でも「誰に給付するか」で乗数が変わる → β（高グループ集中）が優位
問1 解答
減税乗数は c ÷ (1−c)。cX＞cW より X国の減税乗数がW国を上回るため、同額の減税でもX国のGDP増加（ΔYX）はW国（ΔYW）より大きくなる。
ΔYW = [cW÷(1−cW)]×減税額 ＜ ΔYX = [cX÷(1−cX)]×減税額
問2 解答
β（限界消費性向の高いグループのみに2S円給付）の方がGDP増加効果が大きい。
総給付額が等しくても、乗数の大きい（c が高い）家計グループへ集中するβの方が、全体としてのΔYが大きくなる。αはc の高低が混在するため実効乗数が平均値にとどまる。
📖 復習ページ：テキスト p.22〜27（45度線分析・乗数効果）
【章末問題 1.2】 IS-LM分析 ― 財政政策・金融緩和・AD-AS（スタグフレーション） テキスト p.28〜50 ｜ 複数年出題
問1．政府が公共事業などを積極化し支出を増加させると、IS曲線・LM曲線・利子率・GDPにどのような変化が生じるか。図に実線を加えて説明しなさい。
問2．自民党新政権（2012年12月）が問1の支出増加を行い、中央銀行による金融緩和も同時に行われた。①金融緩和の政策手段、②GDPと利子率の変化を問1の図に点線を加えて説明しなさい。
問3．金融緩和で円安が進行し輸入原材料価格が上昇した場合、AD-AS分析を用いて物価とGDPへの影響を説明しなさい。
⚠ 注意（問3）
輸入原材料価格の上昇はAS曲線のみに作用する。AD曲線は変化しない。「スタグフレーション（物価↑＋GDP↓）」のキーワードと論理を明確に述べること。
解答・解説
📌 解くためのポイント
  • 財政拡張 → IS右シフト → GDP↑・金利↑ → クラウディング・アウト
  • 財政＋金融同時 → IS右シフト＋LM右シフト → GDP大幅↑・金利上昇を相殺
  • 輸入原材料↑ → AS左上シフト → スタグフレーション（物価↑・GDP↓）
  • AD曲線は需要サイドの集計。コスト上昇はAS（供給サイド）にのみ影響
問1 解答
政府支出↑ → IS曲線右シフト（IS→IS'）→ LM曲線との交点移動でGDP↑・利子率↑。民間投資が利子率上昇で抑制されるクラウディング・アウト効果が生じる。
問2 解答
①買いオペレーション（公開市場での国債買取）によりマネーストック↑。
②LM曲線が右下方シフト（LM→LM'）。問1で上昇した利子率が低下してクラウディング・アウトを相殺し、GDPは財政のみの場合よりも大幅に増加する。
問3 解答
輸入原材料価格上昇 → 生産コスト↑ → AS曲線左上シフト。AD曲線は変化せず。均衡点が左上に移動し、物価↑・GDP↓。これがスタグフレーション。
輸入原材料価格↑ → 生産コスト↑
AS曲線 左上シフト（AD曲線は不変）
均衡点が左上に移動
物価↑ かつ GDP↓ ＝ スタグフレーション

【章末問題 1.3】 物価指標の識別・AD-AS分析・フォワードガイダンス・フィッシャー方程式 テキスト p.11〜21, p.40〜54, p.113 ｜ 近年出題
問1．3本の折れ線（a）（b）（c）について消費者物価指数・GDPデフレーター・国内企業物価指数のどれか答えなさい。また2021年に(a)が低下し(b)が上昇した理由を、原油価格動向を参照して説明しなさい。
問2．原油価格上昇が実質GDPに与える影響をAD-AS曲線で検討しなさい（名目賃金は硬直的・非自発的失業あり）。(1)各軸・各曲線の意味を記しなさい。(2)原油価格上昇による生産コスト上昇の実質GDPと物価への影響を説明しなさい。
問3．日銀が導入したフォワードガイダンスについて：(1)定義を簡潔に説明しなさい。(2)フィッシャー方程式に基づき予想物価上昇率を名目金利と実質金利で示しなさい。(3)予想物価上昇率の上昇がGDPに影響する波及経路を2つ挙げなさい。
⚠ 注意（問1のひっかけ）
GDPデフレーターは「国内産出の付加価値の価格」。輸入品は含まない。よって輸入価格上昇がGDPデフレーターを「低下」させるメカニズムを論理的に説明すること。
解答・解説
📌 解くためのポイント
  • GDPデフレーター：国内付加価値の価格（輸入含まない）→ 原油↑で低下しうる
  • CPI（消費者物価指数）：小売段階の財サービス価格。輸入品含む → 原油↑で上昇
  • 国内企業物価指数：企業間取引財の価格。原油等資源価格の変動に最も敏感で変動幅大
  • フィッシャー方程式：名目金利 ＝ 実質金利 ＋ 期待インフレ率 → 予想物価↑ → 実質金利↓
問1 解答
(a)：GDPデフレーター　(b)：消費者物価指数　(c)：国内企業物価指数
理由：原油価格上昇期（2021年）に(c)が最大変動 → 企業間財価格の国内企業物価指数。(a)は低下 → 輸入コスト上昇が国内付加価値価格を押し下げるGDPデフレーター。(b)は上昇 → 小売価格に転嫁されるCPI。
問2 解答
(1) 縦軸：物価(P)、横軸：実質GDP(Y)、右下がり曲線：AD曲線（総需要曲線）、右上がり曲線：AS曲線（総供給曲線）
(2) 原油価格↑ → 生産コスト↑ → AS曲線左上シフト。AD曲線不変。均衡点が左上移動 → GDP↓・物価↑（スタグフレーション）
問3 解答
(1) 中央銀行が政策目標実現まで（またはある期間）金融緩和を継続するとコミットメントする政策
(2) 予想物価上昇率 ＝ 名目金利 − 実質金利
(3) ①予想物価↑ → 実質金利↓ → 設備投資↑ → 総需要↑ → GDP↑
　　②予想物価↑ → 実質金利↓ → 円安 → 輸出↑ → 総需要↑ → GDP↑

【章末問題 1.4】 AD-AS分析（名目賃金上昇）・マンデル＝フレミングモデル（変動相場制） テキスト p.40〜50, p.196〜207 ｜ 近年出題
問1．名目賃金の上昇がマクロ経済に与える影響をAD-ASモデルで分析しなさい。
(1) 名目賃金の上昇だけを考慮した場合の物価(P)とGDP(Y)への影響
(2) 名目賃金上昇に加えて民間消費の増加も考慮した場合の影響
問2．以下の空欄①〜④を答えなさい。大国の金融引締めにより、小国のBP曲線が（①）にシフトする。このとき小国通貨が（②）するので、小国の（③）曲線は右方にシフトし、小国のGDPは（④）する。（小国：変動相場制・資本移動完全）
⚠ 注意（問1(2)のひっかけ）
名目賃金上昇＋民間消費増加の複合効果：GDPが増加か減少かは「AD曲線のシフト幅がAS曲線のシフト幅を上回るかどうか」に依存する。「GDPは増加する」と断言せず、条件付きで述べること。
解答・解説
📌 解くためのポイント
  • 名目賃金↑：生産コスト増 → AS左上シフト（AD曲線には直接影響しない）
  • 民間消費↑：需要増 → AD右上シフト。ASとADの同時シフトでGDP変化は不確定
  • マンデル＝フレミング（変動相場・完全資本移動）：大国金利↑ → 小国から資本流出 → 小国通貨減価 → 輸出↑ → IS右シフト → GDP↑
  • 変動相場制では金融政策有効・財政政策無効（IS-LMとの違いを意識）
問1(1) 解答
名目賃金↑ → 生産コスト↑ → AS曲線が左上シフト。AD曲線は変化しない。均衡点が左上に移動し、GDPは減少・物価は上昇する。
問1(2) 解答
名目賃金↑ → AS左上シフト、民間消費↑ → AD右上シフト。物価は確実に上昇。GDPの変化は不確定であり、AD曲線の右シフトがAS曲線の左シフトを上回れば増加し、そうでなければ減少する（均衡点がE₀の右方に移動するかどうか次第）。
問2 解答
①上方　②減価（安くなる）　③IS　④増加
マンデル＝フレミングの論理（変動相場・完全資本移動）
大国が金融引締め → 大国の金利↑
BP曲線が上方シフト（世界金利上昇）
小国から大国へ資本流出 → 小国通貨が減価
小国の輸出↑・経常収支改善 → IS曲線が右シフト
小国のGDP増加（逆説的な結果）
